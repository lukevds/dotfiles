The architecture of this computer is x86_64 and I use it with the operating system and package manager [[https://guix.gnu.org/][GNU Guix]] and with the symlink manager Stow. To install the stow package containing configuration for Emacs, i3 window manager and i3 status, issue the following command on your system:
#+begin_src bash
  stow -t ~ home
#+end_src
If you want to uninstall it:
#+begin_src bash
  stow -t ~ -D home
#+end_src

* Operating System
The operating system configuration of this computer is available at [[./spinfusor/system.scm][spinfusor/system.scm]]. You can create your own config based on it using:
#+begin_src scheme
  ; my-config.scm
  (use-modules (spinfusor system))

  (operating-system
    (inherit os)
    ...)
#+end_src
And then install it using:
#+begin_src bash
  sudo GUILE_LOAD_PATH=$GUILE_LOAD_PATH:${PWD} guix system reconfigure my-config.scm
#+end_src
